    # chartData格式声明


    1. 参数options：可以定制化业务options需求，和官网参数一样
    2. 参数xData（Array）: 传名称的数组集合（如果yData是使用对象的格式，则xData非必传）
    3. 参数yData（Array）: 和series里面的data属性对应，可以有两种格式（具体看官网） 
    // 形式一
chartData= {
    xData: ["直接访问", "邮件营销", "联盟广告", "视频广告", "搜索引擎"],
    yData: [335, 310, 234.135, 1848, 231, 789, 635],
}

// 形式二
chartData= {
    // 自定定义的options
    options: {
        tooltip: {
            show: false,
        },
    },
    toUnitFn: bytesToSize, // 把tooltip数值转换单位
    // 对象的格式
    yData: [
        { value: 50, name: "直接访问" },
        { value: 310, name: "邮件营销" },
        { value: 204, name: "联盟广告" },
        { value: 405, name: "视频广告" },
        { value: 1548, name: "搜索引擎" },
    ],
}
