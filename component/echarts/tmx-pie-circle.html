    # 使用

    # 1. 示例一

    1.TmxPieCircle 组件包括 echart 饼图和独立的图例，所以需要自己调宽高和边距 padding

    2.设置了(size)固定左边的大小，右边的图例会自适应填满,并且每列平均分

    Total

    360

    • 直接访问

      50 13.89%

    • 邮件营销

      310 86.11%

    <template>
    <PieBar :chartData="pieData" style="border: 1px solid;padding: 40px 24px 0 10px;width: 392px; height: 260px" :size="124" :pageSize="6"></PieBar>
</template>


    export default {
  data() {
    return {
      pieData: {
        yData: [
          { value: 50, name: "直接访问" },
          { value: 310, name: "邮件营销" },
        ],
      },
    };
  },
};
