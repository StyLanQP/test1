# 安装

使用 npm:

npm install tmx-ui --save

或者 yarn:

yarn add tmx-ui -D

# 使用

全局使用:

import Vue from 'vue'
import TmxUI from 'tmx-ui'
Vue.use(TmxUI)

或者按需加载（建议使用这种方式）:

import { TmxCard } from 'tmx-ui'
export default {
  components: {
    TmxCard,
  },
}

# 例子

<template>
  <TmxCard />
</template>

# 部署 & 构建

# docs

该项目配置相关的组件说明文档，你可以在这里（/docs）编写你的文档，同时还可以作为开发环境，实时查看你的组件。文档相关部署如下：

# 文档本地启动
yarn docs:dev

# 文档打包
yarn docs:build

# 文档部署
yarn deploy:build

如果你不喜欢这种实时文档的开发模式，该项目还支持传统的开发模式，你可以在这里（/example）引用你的组件进行调试。

yarn dev

# build

你可以通过以下命令对组件库进行打包：

yarn build:prod

或样式构建：

yarn build:style

同时，你也可以单独打包样式和组件：

yarn build:component

如果你觉得以上命令太麻烦、太复杂，你也可以选择一键部署：

yarn build
