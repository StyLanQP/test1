# 更新

# 说明 :

1.5版本完成了iview的基础组件抽离
从 1.4 更新至 1.5 可以体验更轻便的TmxUI库

# 使用 :

  import TmxUI from 'tmx-ui'

全局注册指令: TmxUI.TmxUiDirective.importDirective
全局挂载弹框: TmxUI.ModalFunction

可删除按需引入: eg: import { TmxCard } from 'tmx-ui'
可替换iview手动引入: 
  eg: import zhCnLocale from 'view-design/src/locale/lang/zh-CN'
    -> import zhCnLocale from 'tmx-ui/src/locale/lang/zh-CN'
  eg: import { oneOf } from 'view-design/src/utils/assist';
    -> import { oneOf } from 'tmx-ui/src/utils/assist';

可删除iview样式引入: 1.5.13+
  import 'view-design/dist/styles/iview.css'
  -> 替换为 import 'tmx-ui/lib/styles/tmx-ui-view.css'
