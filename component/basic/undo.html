    # Carousel 走马灯

    # 概述

    常用于一组图片或卡片轮播，当内容空间不足时，可以用走马灯的形式进行收纳，进行轮播展现。

    # 添加属性：

    scale: （原子项宽度是默认取Carousel的宽度）scale为倍数，默认值为1，添加scale后可任意取默认宽度的倍数。

    # Message 全局提示

    # 概述

    轻量级的信息反馈组件，在顶部居中显示，并自动消失。有多种不同的提示状态可选择。

    # 全局配置方法：

    this.$Message.config({
    top: 50,
    duration: 3,
    iconFontSize: '15',
    isUseTmxIcon: false,
    iconPosition: 'start',
    tmxIconTypes: {
        'info': 'Info',
        'success': 'Correct',
        'warning': 'Alarm',
        'error': 'CloseCircle',
    }
});
