    # 折叠面板 Collapse

    # single类型

    header圆角， 内容带边框

    Accordion on the page is in primary status.

    具有副标题的折叠面板

    Accordion on the page is in primary status.

    Child of the first accordion. Child of the first accordion.

    代码示例 
    <template>  
    <Collapse type="single" v-model="value1" style="margin-top: 30px;">
        <h4 style="margin: 32px 0 16px;">header圆角， 内容带边框</h4>
        <Panel name="3" >
            Accordion on the page is in primary status.
            <Button style="float:right;margin:4px 7px;" class="tmx-btn" type="primary">Save</Button>
            <p slot="content" style="padding:16px;">内容</p>
        </Panel>
        <h4 style="margin: 32px 0 16px;">具有副标题的折叠面板</h4>
        <Panel name="4" size="large">
            <div class="title" style="display: flex;">
                <div style="flex:1">
                    <p style="line-height: 21px;margin-bottom: 4px;">
                        Accordion on the page is in primary status.
                    </p>
                    <p style="color: #6B7A99;line-height: 15px;font-size:12px;">
                        Child of the first accordion. Child of the first accordion. 
                    </p>
                </div>
                <Button style="margin-top: 4px;" class="tmx-btn" type="primary">Save</Button>
            </div>
            <p slot="content" style="padding:16px;">内容</p>
        </Panel>
      </Collapse>
</template>

                    第二层

                            title

                        </Panel>

                </Panel>
            </Collapse>
        </Panel>
    </Collapse>
</template>
