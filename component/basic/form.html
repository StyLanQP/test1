    # 表单 Form

    1. 录入控件与标题之间的间距为 4px，2 个 box 之间的上下间距固定为 24px，左右间距固定为 32px；
    2. 表单中 box 的宽度需要自适应可显示范围的宽度，最多自适应到 1920 的分辨率，之后将不会继续变宽；
    3. 设计稿给予宽度为屏幕最小宽度 1440px 时的宽度（在对话框 / 抽屉内时因其宽度固定，不用适配浏览器）；
    4. 当 box 的下方存在 Checkbox 或 Switch 时，他们之间的上下间距为 8px；
    5. Radio 或 Checkbox 内选项之间的间距固定为 24px、与标题之间的上下间距为 18px；
    6. 错误提示固定出现在 box 的下方，离 box 的间距为 2px。当 box 下方存在 Checkbox 或 Switch 时，需要出现在二者中间（详情看图例）；
    7. form label 添加tooltip的使用方法同iview一样添加slot，只是在tooltip组件上添加一个tmx-form-label-tooltip类来规范icon的样式；
