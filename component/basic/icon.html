    # Icon 替换方法

    需要UI打包下载所有icon的文件夹，然后进行替换

    步骤1.覆盖src\styles\common\fonts下 iconfont. css\svg\ttf\woff\woff2 等五个文件

    步骤2.将iconfont.css文件下所有.icon-*样式复制粘贴到src\styles\common\icommon.less文件下，

    步骤3.将iconfont.css文件下@font-face.src路径复制粘贴到src\styles\common\icommon.less文件下，并修改为正确路径（加个fonts/)

    步骤4.将icommon.less文件中兼容样式content改为正确的content，在icommon.less文件最底部

    步骤5.将新增的icon名称添加到docs\scripts\icon.js文件中,用于文档展示
