    # Slot 弹框组件-提示
    Warning !
    Do you want to save your changes?
    <TmxModal
  :icon= "{ size: '21', color: '#0A8BF5', type: 'Alarm'}"
  :is-show-leave.sync="isShowLeave3"
  :save-loading="loading"
  :btn-opts="btnOpts"
  content="Do you want to save your changes?"
  @on-leave-cancel="leaveCancelHandler"
  @on-leave-discard="leaveDisacrdHandler"
  @on-leave-save="leaveSaveHandler"
>
  <!-- <div slot='tmxheader'>头部</div>
    <div slot='tmxcontent'>内容部位</div>
    <div slot='tmxfooter'>底部</div> -->
</TmxModal>
btnOpts:{ save:'保存',//不传不显示该按钮  discard:'忽略'，cancel:'取消' }

函数调用：
1、使用方法类似iview的
2、参数有title icon content btnOpts saveLoading onCancel onDiscard onOk onClose
3、全局需注册TmxConfirm：
    import {ModalFunction} from 'tmx-ui'
    Vue.prototype.$Modal.TmxConfirm = (options) => {
        let modal = ModalFunction()
        Vue.prototype.$Modal.removeTmxConfirm = modal.remove
        return modal.confirm(options)
    }
4、异步确认后需要 手动关闭弹框 this.$Modal.removeTmxConfirm()
5、tip：按钮位置是固定的、并且暂不支持slot，如需slot请使用TmxModal组件

eg:    
this.$Modal.TmxConfirm({
    title: this.t('warning'),
    content: this.t('devices.message_reset'),
    icon: { size: '21', color: '#0A8BF5', type: 'Alarm'},
    btnOpts: {
        save: this.t('ok'),
        discard: this.t('cancel'),
        cancel: this.t('cancel'),
    },
    onOk: async () => {
        this.loading = true
        this.getActionRequest(params)
        this.$Modal.removeTmxConfirm()
    }
})
