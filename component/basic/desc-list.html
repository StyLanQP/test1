    # 描述列表 Desc List

    # 说明

    每个单元格都是该数据的可视范围宽度，需要给每个单元格一个默认宽度（也是最小宽度，设计稿的宽度既是默认宽度） 载入时，当一行中所有单元格的默认宽度相加小于可显示区域时，将多出部分 px 平均分配给标题对应内容的单元格（标题固定宽度） 单元格的高度均为固定高度，一行的高度 32px，需要多行显示数据时，单元格的高度以设计稿为准 当内容超出可视范围时，一行的单元格省略内容 Tips 显示所有。多行的单元格换行显示内容，超出单元格高度时，单元格内出现滚动条
