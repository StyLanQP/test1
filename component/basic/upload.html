    # 上传 Upload

    # 基础用法

    任意格式文件上传都可以使用该样式，可删除已上传文件，或直接点击按钮重新上传。文件名称展示的宽度固定为 320px，文本显示不全则直接省略即可，Hover 显示全称。

    * Only accept .pdf .jpg file under 6M
      <template>
  <Upload
    class="tmx-upload"
    multiple
    action="http://dev.thingsmatrix.cn/api/file/upload/image?type=model"
  >
    <button class="tmx-btn">Select File</button>
    <span class="upload-tips">* Only accept .pdf .jpg file under 6M</span>
  </Upload>
</template>

      # 图片上传

      图片格式的文件上传，上传成功后，会显示图片的缩略图。当悬浮在缩略图的范围内，会出现删除的 Icon。单图片上传也可以通过上传覆盖替换图片，不用先删除再上传。

      <template>
  <Upload
    class="tmx-img-upload"
    ref="upload"
    :show-upload-list="false"
    :format="['jpg','jpeg','png']"
    :max-size="2048"
    type="drag"
    action="http://dev.thingsmatrix.cn/api/file/upload/image?type=model"
  >
    <div class="tmx-img-upload-content">
      <TmxIcon type="Camera" size="24"></TmxIcon>
    </div>
  </Upload>
</template>
