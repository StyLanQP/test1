    # 基础用法 默认

      draggable
      for
      vue.js 2.0
      based
      on
      Sortablejs
      vue.draggable
      component
    <template>
  <TmxDragger 
    :list="list"
    tag="ul"
    class="list-group"
    handle=".handle"
    @on-drag-change="onDragChange"
    @on-move="onMove"
  />
</template>

    .sortable-chosen{
  border: 1px solid var(--c-primary08);
}

     onMove(isMove, futureIndex, e){
    console.log('isMove: ', isMove)
    console.log('futureIndex', futureIndex) // 鼠标拖拽位置对应的元素下标
  },

    # 说明

    通过类名 sortable-chosen 开发当前悬浮拖拽项的样式
    通过类名 tmx-dragger-source-item 开发选中项的样式，没有样式设置时，则继承 sortable-chosen 样式

    # Swap模式

      draggable
      for
      vue.js 2.0
      based
      on
      Sortablejs
      vue.draggable
      component
    <template>
  <TmxDragger 
    swap
    :list="list"
    tag="ul"
    class="list-group"
    handle=".handle"
    @on-drag-change="onDragChange"
    @on-move="onMove"
  />
</template>

    # 说明

    通过设置 swap 属性，开启swap模式
    类表格用法暂不支持swap模式

    # slot用法

      handle draggable
      handle for
      handle vue.js 2.0
      handle based
      handle on
      handle Sortablejs
      handle vue.draggable
      handle component 
    <template>
  <TmxDragger 
    :list="list"
    tag="ul"
    class="list-group"
    handle=".handle"
    ghostBorder
    @on-drag-change="onDragChange"
    @on-move="onMove"
  >
    <template #list>
      <div v-for="element in list" :key="element.id" class="item">
        <span class="handle">handle</span>
        {{element.name}}
      </div>
    </template>
  </TmxDragger>
</template>

    # 说明

    使用slot开发自定义样式，slot名为list。

    <template>
  <TmxDragger 
    :list="listTable"
    :columns="columns"
    tag="ul"
    class="list-group"
    handle=".handle"
    type="table"
    cellWidth="260px"
    @on-drag-change="onDragChange"
    @on-move="onMove"
  >
    <template slot="action" slot-scope="{ row, index }">
        <Button type="primary" style="margin-right: 5px">View</Button>
        <Button>Delete</Button>
    </template>
  </TmxDragger>
</template>

    # 说明

    设置 type="table"
    list和columns参数格式同table组件
    可以通过slot自定义列样式，使用方法同table组件
